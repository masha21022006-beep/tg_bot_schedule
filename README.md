# Telegram Schedule Bot

## Краткое описание проекта и цель
Telegram Schedule Bot — учебный проект Telegram-бота для работы с расписанием занятий в университете.  
Цель проекта — продемонстрировать разработку Telegram-бота на Python с поддержкой многопользовательского режима, хранением пользовательских данных и контейнеризацией приложения с помощью Docker.

Бот позволяет каждому пользователю Telegram:
- просматривать своё расписание занятий;
- создавать расписание;
- редактировать существующее расписание.

Учебные дни: **понедельник–пятница**.  
Суббота и воскресенье считаются выходными днями.  
Максимум **4 занятия (пары)** в день.

## Архитектура и структура проекта
Проект реализован как односервисное приложение.

### Общая архитектура
- Telegram Bot API — взаимодействие с пользователями
- Python-скрипт — бизнес-логика бота
- JSON-файл — хранение расписаний пользователей
- Docker — контейнеризация приложения

### Структура проекта
Schedule_bot_unlim.py # основной файл Telegram-бота
Dockerfile # инструкция сборки Docker-образа
docker-compose.yml # запуск контейнера
requirements.txt # зависимости Python
.gitignore # исключения Git
README.md # документация проекта

Файлы `.env` и `schedules.json` в репозитории отсутствуют и создаются пользователем локально.

### Примеры использования
Вход
Пользователь запускает бота в Telegram командой:
/start
Основные действия пользователя
Просмотр расписания — выбор дня недели
Создание расписания — заполнение занятий по дням
Редактирование расписания — изменение ранее введённых данных
Взаимодействие с ботом осуществляется через кнопочное меню (Inline Keyboard).
HTTP-эндпоинты и CLI-аргументы не используются, так как проект является Telegram-ботом.
Выход
Специальная команда выхода не требуется. Пользователь может в любой момент прекратить диалог с ботом.
